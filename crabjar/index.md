# Crabjar Workspace
The agent's cognitive workspace for the Machine-Assisted Learning (MAL) loop.

**Key Files:** `src/main.rs`, `state-docs/`, `memory/`, `reference_materials/`
**Dependencies:** `crabjar-config`, `agent-context`, `mirror-guard`
**Data Source:** SQLite-backed memory and filesystem state-docs
**Capabilities:** State-docs read/write, knowledge store queries, command dispatch
